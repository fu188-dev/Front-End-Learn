应用样式和色彩

色彩

fillStyle = color
设置图形的填充颜色。默认颜色为黑色#000000。颜色值还可以是渐变或者图案
strokeStyle = color
设置图形轮廓的颜色。默认颜色为黑色#000000。颜色值还可以是渐变或者图案

颜色值如下：

ctx.fillStyle = "orange";

ctx.fillStyle = "#FFA500";

ctx.fillStyle = "rgb(255,165,0)";

ctx.fillStyle = "rgba(255,165,0,1)";

透明度

globalAlpha = transparencyValue
这个属性影响到 canvas 里所有图形的透明度，和 CSS 里面的 opacity 属性一样。
值的范围是 0.0（完全透明） ~ 1.0（完全不透明），默认是1.0。
建议用 rgba 进行不透明度的设置

线型

lineWidth = value
设置线条宽度。必须为正数，默认值是 1.0。
就是在路径的两边各绘制线宽的一半，例如给的宽是10，就会两边各自占5。 这样就会导致线条的位置不准确的问题。
lineCap = type
设置线条末端样式。值如下：
butt（默认值）：方角。
round：圆角，末端会多出一截。
square：方角，末端会多出一截。
lineJoin = type
设定线条与线条间结合处的样式。值如下：
miter（默认值）：连接处是尖的，看着棱角分明。也可以说是边缘被延伸交汇于一点上。
round：连接处是圆角的，看着比较舒服。
bevel：连接处是平的。
miterLimit = value
限制当两条线相交时交接处最大长度；所谓交接处长度（斜接长度）是指线条交接处内角顶点到外角顶点的长度。
getLineDash()
返回一个包含当前虚线样式，长度为非负偶数的数组。
setLineDash()
设置当前虚线样式。
lineDashOffset = value
设置虚线样式的起始偏移量。
