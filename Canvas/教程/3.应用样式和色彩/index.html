应用样式和色彩

色彩

fillStyle = color
设置图形的填充颜色。默认颜色为黑色#000000。颜色值还可以是渐变或者图案
strokeStyle = color
设置图形轮廓的颜色。默认颜色为黑色#000000。颜色值还可以是渐变或者图案

颜色值如下：

ctx.fillStyle = "orange";

ctx.fillStyle = "#FFA500";

ctx.fillStyle = "rgb(255,165,0)";

ctx.fillStyle = "rgba(255,165,0,1)";

透明度

globalAlpha = transparencyValue
这个属性影响到 canvas 里所有图形的透明度，和 CSS 里面的 opacity 属性一样。
值的范围是 0.0（完全透明） ~ 1.0（完全不透明），默认是1.0。
建议用 rgba 进行不透明度的设置

线型

可以看这里的解析

lineWidth = value
设置线条宽度。必须为正数，默认值是 1.0。
就是在路径的两边各绘制线宽的一半，例如给的宽是10，就会两边各自占5。 这样就会导致线条的位置不准确的问题。
lineCap = type
设置线条末端样式。值如下：
butt（默认值）：方角。
round：圆角，末端会多出一截。
square：方角，末端会多出一截。
lineJoin = type
设定线条与线条间结合处的样式。值如下：
miter（默认值）：连接处是尖的，看着棱角分明。也可以说是边缘被延伸交汇于一点上。
round：连接处是圆角的，看着比较舒服。
bevel：连接处是平的。
miterLimit = value
限制当两条线相交时交接处最大长度；所谓交接处长度（斜接长度）是指线条交接处内角顶点到外角顶点的长度。
getLineDash()
返回一个包含当前虚线样式，长度为非负偶数的数组。
setLineDash(segments)
设置当前虚线样式。
segments: 一个Array数组。一组描述交替绘制线段和间距（坐标空间单位）长度的数字。如果数组元素的数量是奇数，数组的元素会被复制并重复。例如， [5, 15, 25] 会变成 [5, 15, 25, 5, 15, 25]。
lineDashOffset = value
设置虚线样式的起始偏移量。
正数就是向左偏移，负数是向右偏移

渐变

createLinearGradient(x1, y1, x2, y2)
创建线性渐变。
参数表示渐变的起点（x1，y1）和终点（x2， y2）。
createRadialGradient(x1, y1, r1, x2, y2, r2)
创建径向渐变。
前三个定义一个以 (x1,y1) 为原点，半径为 r1 的圆，后三个参数则定义另一个以 (x2,y2) 为原点，半径为 r2 的圆。
gradient.addColorStop(position, color)
给渐变上色。
position：0.0 ~ 1.0之间的一个数值！表示渐变中颜色所在的相对位置。例如，0.5 表示颜色会出现在正中间。
color：颜色值，#FFF、rgba等等。
你可以添加任意多个的色标，例如：
var lineargradient = ctx.createLinearGradient(0, 0, 150, 150);
lineargradient.addColorStop(0, "white");
lineargradient.addColorStop(1, "black");

图案样式 Patterns

createPattern(image, type)
image：可以是一个 Image 对象的引用，或者是另一个 canvas 对象。
type：repeat、repeat-x、repeat-y、no-repeat
用 canvas 对象作为 Image 参数在 Firefox 1.5（Gecko 1.8）中是无效的。

阴影

shadowOffsetX = float
阴影在 x 轴的延伸距离。默认为0，正数向右延伸，负数向左延伸。
shadowOffsetY = float
阴影在 y 轴的延伸距离。默认为0，正数向下延伸，负数向上延伸。
shadowBlur = float
设定阴影的模糊程度。默认为 0。值越大，越模糊。
shadowColor = color
设定阴影的颜色，默认是全透明的黑色。

Canvas 填充规则

当我们用到 fill（或者 clip 和 isPointinPath ） 你可以选择一个填充规则，该填充规则根据某处在路径的外面或者里面来决定该处是否被填充，
这对于自己于自己路径相交或者路径被嵌套的时候是有用的。
有如下两个值：
nonzero：默认值。填充路径的内部
evenodd：填充多个路径重合但未重叠的部分。
你的浏览器不支持 canvas，请更换浏览器使用。
