绘制文本

绘制文本

fillText(text, x, y[, maxWidth])
在指定的 x,y 处填充指定的 文本 。绘制的最大宽度是可选的。
如果设置了最大宽度，所有内容将会填充到宽度之内，不会超出此范围。
strokeText(text, x, y[, maxWidth])
在指定的 x,y 处绘制 文本边框 。绘制的最大宽度是可选的。
如果设置了最大宽度，所有内容将会填充到宽度之内，不会超出此范围。
你的浏览器不支持 canvas，请更换浏览器使用。

有样式的文本

font = value
和 CSS 的 font 语法相同。默认是：10px sans-serif。
textAlign = value
文本对齐方式。值包括：
start：默认值。
end
left
right
center
textBaseline = value
基线对齐方式。值包括：
alphabetic：默认值。
top
hanging
middle
bottom
direction = value
文本方向。值包括：
inherit：默认值。
ltr
rtl
你的浏览器不支持 canvas，请更换浏览器使用。

预测量文本宽度

measureText(text)
将返回一个 TextMetrics 对象。
text：需要测量的字符串。
TextMetrics 对象会返回如下属性（属性都是只读的）：
width：字符串的宽度。double 类型。
actualBoundingBoxLeft：textAlign 标明的对齐点到文本矩形边界左侧的距离。double 类型。
actualBoundingBoxRight：textAlign 标明的对齐点到文本矩形边界右侧的距离。double 类型。
fontBoundingBoxAscent：textBaseline 标明的基线到到渲染文本的所有字体的矩形边界最顶部的距离。double 类型。
fontBoundingBoxDescent：textBaseline 标明的基线到到渲染文本的所有字体的矩形边界最底部的距离。double 类型。
actualBoundingBoxAscent：textBaseline 标明的基线到渲染文本的矩形边界顶部的距离。double 类型。
actualBoundingBoxDescent：textBaseline 标明的基线到渲染文本的矩形边界底部的距离。double 类型。
emHeightAscent：textBaseline 标明的基线到线框的 em 方块顶部的距离。double 类型。
emHeightDescent：textBaseline 标明的基线到线框的 em 方块底部的距离。double 类型。
hangingBaseline：textBaseline 标明的基线到线框的 hanging 基线的距离。double 类型。
alphabeticBaseline：textBaseline 标明的基线到线框的 alphabetic 基线的距离。double 类型。
ideographicBaseline：textBaseline 标明的基线到线框的 ideographic 基线的距离。double 类型。
你的浏览器不支持 canvas，请更换浏览器使用。
